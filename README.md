# Payroll System Project

This is my SDEV268 final project, built using Java Swing and SQLite.

## What It Does

* Secure login for admins and employees
* Full employee management (with add, edit, delete, search)
* Time entry with paid time off support
* Complete payroll calculations (including overtime, taxes, deductions, and employer costs)
* Payroll locking and report generation
* Bonus features like binary search examples (both recursive and iterative)

## Prerequisites

Before running the project, you must have the following:

1. **VS Code (latest version recommended)**
2. **Extension Pack for Java (in VS Code)**
3. **JDK (17+ recommended)**

## How to Run

### Option 1: Run the batch file

1. Simply double-click the file **payroll.bat** in the project folder. It should compile and launch automatically.

### Option 2: Open in VS Code

1. Open the project folder in VS Code
2. Open the file `src/payroll/Main.java`
3. Click the **Run** button above explorer, or press F5
